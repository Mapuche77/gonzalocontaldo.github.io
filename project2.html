Plane Crash Data Visualization Overview:

This collection of Python scripts aims to visualize historical plane crash data, providing insights into crash patterns over time and geographical locations. The process involves web scraping to gather data, cleaning and enhancing the information, and finally, creating an interactive visual representation.

Modules:

scraper.py: Web Scraping for Data Collection
The scraper.py module extracts relevant plane crash data from the PlaneCrashInfo website. Utilizing the requests library for HTTP requests and BeautifulSoup for HTML parsing, the script compiles a dataset including crash dates, locations, and aircraft details. User input allows selecting data for a specific year.

data_cleaner.py: Data Enhancement and Geocoding
The data_cleaner.py module focuses on refining the extracted data. It employs the geopy library for geocoding location names and removes unnecessary words from the locations. The resulting coordinates, along with corresponding aircraft information, are appended to the dataset. This step is crucial for accurate geographical representation in the visualization.

visualizer.py: Data Visualization with Matplotlib
The visualizer.py module takes the enhanced data and creates a dual-plot visualization using Matplotlib. The left plot displays the distribution of plane crashes across months, providing insights into monthly patterns. The right plot employs Cartopy for geographical mapping, showing red dots representing crash locations. The interactivity is enhanced using mplcursors, allowing users to hover over the dots to view detailed aircraft information.

Instructions:

Run scraper.py to collect data.
Execute data_cleaner.py for data enhancement and geocoding.
Finally, run visualizer.py to generate an insightful visual representation of historical plane crash data.
